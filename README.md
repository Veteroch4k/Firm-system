# Информационная система производственной фирмы
### Назначение

  Данный проект представляет собой информационную систему, предназначенную для автоматизации и управления производственными процессами на предприятии, которое самостоятельно производит товары, ранее закупаемые у сторонних поставщиков. Система отслеживает полный цикл производства: от планирования операций и потребления материалов до учета готовой продукции и контроля использования инструментов.
  
### Ключевые функциональные области

* **Управление инструментами**
* **Управление материалами**  
* **Управление продукцией** 
* **Управление операциями**
* **Управление нарядами**
* **Управление цехами**
